Estadstica descriptiva

Estadstica descriptiva o exploratoria

  • Tradicionalmente se ha entendido como un conjunto de herramientas, formado por coeficientes y tcnicas, que tratan de resumir la informacin contenida en un conjunto de datos.

  • Se constituye como parte fundamental de cualquier anlisis estadstico complejo, en el que se empiezan a tomar decisiones que afectaran el problema de inters.

  • Proporciona informacin sobre la estructura de la poblacin que se estudia, indicando, por ejemplo, si sta es simtrica, si realmente se trata de una nica poblacin o hay una superposicin de poblaciones, deteccin de valores atpicos, etc.

Organizacin de la informacin

  • Los datos constituyen la materia prima de la Estadstica, logrando establecer distintas clasificaciones en funcin de la forma en que stos vengan dados.

  • Se obtienen datos al realizar cualquier tipo de prueba, experimento, valoracin, medicin, observacin.

Bases de datos

Bases de datos

Modelos

Un modelo de datos es una serie de conceptos que puede utilizarse para describir un conjunto de datos y las operaciones para manipularlos. Algunos de los modelos de datos son:

  - Base de datos jerarquica
  - Base de datos relacional    
  - Base de datos de red
  - Base de datos entidad - relacin
  - Base de datos multidimensional
  - Base de datos orientada a objetos

Matriz de datos

  • Al conjunto de individuos fsicos considerados en un anlisis se le denomina Colectivo o Poblacin.

  • Tambin se utilizarn esos mismos trminos para referirse a la(s) caracterstica(s) de esos individuos que son objeto de estudio.

  • Matriz de datos: Los datos son organizados en una matriz de \(n*p\) donde cada fila representa a un individuo o registro y las columnas a las caractersticas observadas.

Manipulacin de datos con R

Importar datos en R

  • Lectura de datos en R:
# Formato "csv"
denPoblac <- read.csv(file = "DenPobla05_10.csv", header = TRUE, sep = ",", dec = ".")

# Formato "csv2"
huevos <- read.csv2(file = "Huevos.csv", header = TRUE, sep = ";", dec = ",")

# Formato "txt"
olimpicos <- read.table(file = "ColOlimpicos.txt", header = TRUE, sep = ",", dec = ".")

Exportar datos en R

  • Guardar conjunto de datos en el directorio de trabajo:
# Formato "csv"
write.csv(denPoblac, file = "DensidadPoblacional.csv")

# Formato "csv2"
write.csv2(olimpicos, file = "ProdHuevos.csv")

# Formato "txt"
write.table(olimpicos, file = "Olimpicos.txt")

Biblioteca tidyr

  • Usos de la biblioteca tidyr:

    • gather(): pasar de formato ancho a largo.
    • spread(): pasar de formato largo a ancho.

  • Verbos tiles de la biblioteca dplyr:

    • select(): selecciona columnas.
    • filter(): selecciona filas.
    • arrange(): reordena filas.
    • mutate(): crea o modifcia columnas.
    • group_by(): operaciones por grupo.
    • summarise(): resumen de datos.
