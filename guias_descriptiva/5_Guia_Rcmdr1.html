Instalar y abrir R Commander

Instalar la biblioteca Rcmdr:

  • En R o RStudio instalar la biblioteca Rcmdr, ejecutando el siguiente comando:
# install.packages("Rcmdr")

Nota: No olvidar instalar las dependencias que requiere la biblioteca Rcmdr.

  • Para abrir R Commander, ejecutar el siguiente código en R o RStudio (en este caso se hará desde R).
# require(Rcmdr)

Bases de datos

Desde R Commander es posible:

  • Generar o crear bases de datos.
  • Manipular bases de datos de R.
  • Importar o leer bases de datos en diferentes formatos (xlx, csv, txt, entre otras)

Gráfico de barras en R Commander

Dirigirse a la opción Gráficas y seleccionar el tipo de gráfico a realizar, en este caso Gráfico de barras

Automáticamente se abrirá una nueva ventana de R donde mostrará el gráfico.

Gráfico de barras con código

barplot(table(iris$Species), xlab="Species", ylab="Frequency")

Consideraciones finales

Existe la posibilidad de crear gráficos de barras desde R Commander, como herramienta de visualización de variables cualitativas o categóricas.

pie(table(iris$Species), xlab="Species", ylab="Frequency")
