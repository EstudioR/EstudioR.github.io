Instalar y abrir R Commander

Instalar la biblioteca Rcmdr:

  • En R o RStudio instalar la biblioteca Rcmdr, ejecutando el siguiente comando:
# install.packages("Rcmdr")

Nota: No olvidar instalar las dependencias que requiere la biblioteca Rcmdr.

  • Para abrir R Commander, ejecutar el siguiente cdigo en R o RStudio (en este caso se har desde R).
# require(Rcmdr)

  • Generar o crear bases de datos.

Grfico de barras en R Commander

Dirigirse a la opcin Grficas y seleccionar el tipo de grfico a realizar, en este caso Grfico de barras

Automticamente se abrir una nueva ventana de R donde mostrar el grfico.

Grfico de barras con cdigo

barplot(table(iris$Species), xlab="Species", ylab="Frequency")

Consideraciones finales

Existe la posibilidad de crear grficos de barras desde R Commander, como herramienta de visualizacin de variables cualitativas o categricas.

pie(table(iris$Species), xlab="Species", ylab="Frequency")
